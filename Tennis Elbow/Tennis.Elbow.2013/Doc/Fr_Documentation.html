 I. Introduction 

Tennis Elbow est un jeu de tennis, à la jouabilité fun et réaliste, simulant le tennis de haut niveau.
Pour configurer les contrôles et la résolution, lancez le raccourci "Configuration" (depuis Démarrer->Programmes->Tennis Elbow 2013).

Configuration minimale requise pour la version Windows :

  • OS: Windows 8/7/Vista/XP/ME/2000/98 (Windows 95/NT non supporté)
  • Processeur: Intel Pentium II à 500 MHz ou équivalent
  • Mémoire: 256 Mo
  • DirectX 8.1
  • Vidéo: carte 3D compatible DirectX 6.0, avec 32 Mo
  • Son: carte compatible DirectX 8.1
  • Clavier, souris, ou joystick
  • 200 Mo disponible sur le disque dur

Configuration recommandée pour la version Windows :

  • Processeur: Intel Pentium III ou AMD Athlon cadencé à 1.2 GHz ou plus
  • Mémoire: 512 Mo ou plus
  • Vidéo: carte 3D compatible DirectX 8.1, avec 64 Mo

Caractéristiques :

Animation :

  • rendu fluide à 60 images per seconde, même sur petit PC
  • 10 animations de coups, dont le revers à 2 mains et le slice
  • mimiques
  • les juges de lignes indiquent les balles fautes et sur la ligne

Tournois :

  • 300 tournois
  • Coupe Davis
  • 3500 joueurs gérés par l'ordinateur, sur 38 ans
  • Double système de classement et gestion des points, en simple comme en double
  • Palmarès, consultation des tableaux de tournois, et de l'historique des joueurs

Divers :

  • 9 terrains, chacun avec un rebond spécifique
  • joueur & joueuse complètement paramétrables, de la couleur des cheveux aux chaussures en passant entre autres par la peau et le t-shirt
  • Simple, jeu à 3, double
  • 6 niveaux de difficulté, chacun découpé en 10 sous-niveaux

 II. Menus 

Dans les menus :

Vous contrôlez la petite balle qui tourne avec la souris ; amenez la balle sur une icône et appuyez sur le bouton n°1 pour activer l'icône; quand la balle est sur une icône, sa description apparait dans le cadre en bas du menu. Vous pouvez également changer l'icône active en utilisant votre clavier ou votre joystick.

Icônes standards :

  • Retour : permet de retourner au menu précédent; l'appui sur le bouton n°2 ou la touche 'Echap' est équivalent à Retour
  • Quitter : permet de retourner sous Windows

Dans de nombreux menus, vous avez une barre d'icônes sur le côté droit. Celles-ci permettent d'accéder aux menus de configuration du profile, des contrôles, du son, de la vidéo, et aux options diverses.

Dans le menu principal :

  • Training Club: pour jouer une partie rapidement et comme vous le souhaitez
  • Compétition : pour participer aux Tournois et être classé
  • Crédits : page d'information

Dans le menu "Training Club" :

  • Jouer : permet de choisir entre faire un échauffement, un match amical, et le mode online, puis lance la partie avec les options en cours
  • Feuilles des joueurs : pour modeler les personnages
  • Options de jeu : pour choisir le terrain, le nombre de sets, le nombre de joueurs et si les changements de côté auront lieu pendant le match ou pas

Dans la feuille de personnage :

  • cliquez sur les caractéristiques pour les faire changer
  • cliquez sur le joueur pour accéder à l'écran de configuration de son apparence
  • cliquez sur l'icône avec une main pour passer de droitier à gaucher
  • Statistiques : permet de voir les statistiques de la carrière du joueur
  • Humain / Ordinateur : pour choisir entre le contrôle par humain et par ordinateur
  • Défenseur / Attaquant de fond de court / Puncheur / Polyvalent / Volleyeur / Contreur : pour choisir le style de jeu
  • Charger / Sauver : pour charger et sauver le joueur
  • Flèches gauche/droite : fait défiler les 4 joueurs ; le 1er joueur est celui en bas de l'écran (par défaut, c'est le joueur que vous contrôlez) ; le 2e joueur est celui en haut de l'écran (par défaut, c'est le joueur contrôlé par l'ordinateur) ; le 3e joueur est le partenaire du 1er joueur pour les doubles, et le 4e est le partenaire du 2e joueur

Dans la configuration de l'apparence :

  • cliquez sur les icônes sur la gauche pour choisir quelle partie configurer, puis réglez les paramètres sur la droite
  • cliquez sur le joueur pour avoir une joueuse, et vice et versa
  • les 2 boutons au milieu en bas vous permettent de copier l'ensemble des paramètres d'apparence ; cliquez sur celui de gauche pour copier le code d'apparence dans le presse papier de Windows (vous pouvez alors utiliser "Edition->Coller" or Ctrl+V pour coller ce code où vous le souhaitez), et cliquez sur celui de droite pour entrer un code précédemment créé (vous pouvez utiliser Ctrl+V pour le rentrer d'un coup)

Dans le menu "Compétition" :

  • Nouvelle partie : pour démarrer une nouvelle partie
  • Continuer la partie en cours : pour continuer la partie en cours qui est sauvegardée quand vous quittez le jeu
  • Charger / Sauver partie : pour charger & sauver vos parties ; si le jeu plante, vous pouvez trouver des sauvegardes automatiques dans le menu de chargement de la partie
  • Match suivant : pour jouer le prochain match, ou pour choisir le prochain tournoi
  • Feuille du joueur : pour observer son personnage
  • Tableaux, classements et scores : pour voir les classements mondiaux, les tableaux des tournois, les résultats de la coupe Davis et les historiques des joueurs
  • Caméra : pour régler tous les paramètres liés à la caméra

Menu "En-Jeu" :

  • appuyez sur "Echappe" pendant l'échauffement ou le match pour accéder au menu En-Jeu
  • Options : permet de configurer les informations affichées

 

 III. Réseau 
(non disponible sur Mac OSX & Linux)

/!\ Important :

  • Si vous voulez jouer sur Internet, vérifiez que vous avez DirectX 9.0c (ou une version plus récente), sinon vous risquez d'avoir des ralentissements monstrueux ! Vous pouvez le télécharger ici : http://www.microsoft.com/downloads/Browse.aspx?displaylang=fr&categoryid=2
  • Si vous êtes derrière un NAT ou routeur, et que vous voulez héberger une partie, vous devez forwarder le port UDP 4321;  si vous avez besoin d'aide, vous trouverez des instructions en anglais ici : http://www.portforward.com/routers.htm ; choisissez votre routeur, puis n'importe quel jeu (par exemple "Quake 3"), cela n'a pas d'importance, il faut juste remplacer le n° de port du jeu par 4321 UDP
  • De nombreux FAI fournissent une Box en mode routeur par défaut. Vous devez alors ouvrir le port UDP 4321.
    Jetez un oeil ici pour savoir comment configurer votre Box : http://www.ratiatum.com/forum/index.php?showtopic=46422
  • Si vous avez un pare-feu (firewall), vous devez ouvrir le port UDP 4321, et autorisez Tennis Elbow à accéder à Internet
  • Si vous pouvez joindre des parties, mais que personne ne joint les parties que vous créez, cela veut probablement dire que vous avez un routeur et que vous ne l'avez pas ou mal configuré

Dans le menu "Réseau" :

  • En haut du menu "Réseau" sont affichées les interfaces TCP/IP détectées par le jeu. Vous pouvez les faire défiler, afin de connaître vos différentes adresses IP, mais ce choix n'a pas d'incidence sur la création de la connexion, qui se fait sur toutes les interfaces disponibles en même temps
  • Port : 4321 par défaut; vous pouvez régler un autre port si le besoin s'en fait sentir, mais le port doit être le même pour l'hôte et le client (sauf si vous faites une connexion à une partie distance en spécifiant une adresse complétée du n° de port, par ex : 192.168.0.1:4000 ); vous pouvez mettre "-1" pour choisir un port au hasard à la création d'une partie, ou rechercher des parties sur tous les ports quand vous joignez une partie distante
  • Héberger une partie : pour créer une partie et attendre qu'un joueur vous rejoigne
  • Joindre une partie : pour chercher des parties sur Internet, sur le réseau local, ou joindre une partie distante
  • Salon de discussion : ouvre le chat de Mana Games, afin de discuter avec d'autres joueurs
  • Forum : ouvre le forum de Mana Games
  • Documentation : ouvre la documentation complète du jeu

Dans le menu "Héberger une partie" :

  • Créer une partie : pour créer la partie avec les paramètres actuels
  • Ping Max : règle le ping maximal que doit avoir un joueur pour joindre votre partie (uniquement pour les parties publiques)
  • Type: mettre "Publique" pour que votre partie soit listée par le serveur global de Mana Games, ou sur "Privé" pour ne pas être listé
  • Permission : mettre "Accepte tout joueur" pour accepter tous les joueurs, et "Seulement version complète" pour accepter seulement les joueurs ayant la version complète de Tennis Elbow
  • Compétences : permet de régler les restrictions sur les compétences, utile pour créer des parties plus équilibrées

Dans le menu "Joindre une partie" :

  • Trouver des parties Internet : recherche toutes les parties Internet listées par le serveur global de Mana Games
  • Joindre une partie locale : scanne les parties disponibles sur votre réseau local (LAN) ; vous devez connaître et spécifier le port du serveur que vous recherchez ; normalement, c'est celui par défaut, c'est-à-dire 4321
  • Joindre une partie distante : pour joindre la partie qui se trouve à l'adresse IP distante spécifiée ci-dessous
  • IP Distante : adresse IP (ou URL) de la partie à rejoindre

Dans le menu "Parties Internet" :

  • Nom des parties : cliquer sur le nom d'une partie pour la rejoindre
  • Rafraîchir les pings : rafraîchit les pings de la liste actuelle
  • Rafraîchir la liste : rafraîchit la liste des parties
  • Cacher les parties non disponibles : mettre à "Oui" pour cacher les parties que vous ne pouvez pas joindre (parce que votre ping est trop élevé, que vous n'avez pas la version complète, etc...)
  • Le ping est une estimation du temps d'aller-retour d'un paquet entre deux ordinateurs. Plus il est petit, plus les communications seront réactives et ainsi le transfert de données sera fluide. Si le ping affiché est "oo", alors la communication n'a pas pu être établie. Cela peut être du à un paquet perdu si cela se produit qu'une seule fois, ou à un parefeu ou un routeur mal configuré, si c'est systématique

Modes de compétence :

  • Libre :
    • il n'y a aucune restriction, toutes les compétences peuvent avoir n'importe quelle valeur de 0 à 100%
  • Fair Play:
    • la visualisation de la visée n'est pas autorisée
    • l'Auto-Positionnement est sur Moyen
    • la moyenne des compétences est de 80%, ce qui signifie que vous devez avoir 780 points disponibles dans votre feuille de personnage
    • les compétences de Service et de Coup Droit sont ré-échelonnées sur 20-70%
    • les compétences de Revers sont ré-échelonnées sur 20-70%
    • toutes les autres compétences sont ré-échelonnées sur 25-75%
  • Réaliste :
    • la visualisation de la visée n'est pas autorisée
    • l'Auto-Positionnement est sur Moyen
    • la moyenne des compétences est de 80%, ce qui signifie que vous devez avoir 780 points disponibles dans votre feuille de personnage
    • toutes les compétences sont ré-échelonnées sur 30-90%

Si vous ne respectez pas les conditions des modes Fair Play ou Réalistes, alors le moteur du jeu ajustera automatiquement vos compétences.

Exemple de ré-échelonnement de compétences :

Valeur Initiale 0% 30% 50% 70% 100%

Résultat pour les compétences en Mode Réaliste

 30% 48% 60% 72% 90%
Résultat pour les compétences de Service & de Coup Droit en Mode Fair Play 20% 35% 45% 55% 70%
Résultat pour les compétences de Revers en Mode Fair Play 15% 30% 40% 50% 65%
Résultat pour les autres compétences en Mode Fair Play 25% 40% 50% 60% 75%

Formules de ré-échelonnement des compétences :

  • Compétences en Mode Réaliste : Compétence en-jeu = 30 + 60 * Compétence / 100
  • Compétences de Service & de Coup Droit en Mode Fair Play : Compétence en-jeu = 20 + 50 * Compétence / 100
  • Compétences de Revers en Mode Fair Play : Compétence en-jeu = 15 + 50 * Compétence / 100
  • Autres compétences en Mode Fair Play : Compétence en-jeu = 25 + 50 * Compétence / 100

Création de partie :

Une fois que vous avez créé une partie et que vous attendez qu'un joueur vous rejoigne, vous pouvez basculer vers une autre application. Tennis Elbow jouera un son d'avertissement dès qu'un joueur rejoindra votre partie.

Sur l'écran suivant, vous devez spécifiez le "Délai" voulu. Si vous n'êtes pas sûr de comment le choisir, mettez-le sur "Auto", patientez 5 secondes, puis cliquez sur "Jouer".

Le Délai est un temps de latence interne qui permet à Tennis Elbow de tourner en parallèle sur votre PC et celui de votre adversaire. Il existe un temps mort de votre visée qui dure la moitié du Délai, juste avant que vous ne frappiez dans la balle. Pendant ce temps mort, le jeu considère que vous continuez de presser gauche (ou droite) si vous étiez en train de presser gauche (ou droite) juste avant le début de ce temps mort, et inversement que vous ne pressez rien du tout si vous n'aviez rien de pressé juste avant le début du temps mort. Concrètement, cela veut dire que si vous pressez gauche pour viser à gauche, et que vous relâchez la pression sur gauche juste avant de frapper la balle, votre visée ne reviendra pas vers le centre comme lorsque vous jouez contre l'ordinateur. Il vous faut donc préparer votre visée au bon moment car vous ne pourrez pas rattraper au dernier moment une visée qui serait allée trop loin, tout spécialement lorsque vous jouez avec un haut Délai. Notez qu'il est recommandé de viser également comme cela contre l'ordinateur, afin d'avoir une visée plus précise.

En résumé : quand le Délai est très haut, le jeu ne détectera pas votre sélection de coup et votre visée si vous ne préparez pas votre coup suffisamment à l'avance.

Pour que le jeu fonctionne à vitesse normale, le Délai doit être au moins la moitié du ping. Par exemple, vous avez généralement au maximum 100ms de ping avec votre adversaire, vous devez donc mettre le Délai à 50ms ou plus. Pour éviter d'avoir des ralentissements ( = lag) pendant vos parties, il est conseillé d'avoir le Délai égal au ping, voir plus haut.

Rejoindre une partie :

Quand vous joignez une partie, vous avez juste à attendre que l'hôte règle le Délai et lance le jeu. Pour connaître le Délai réglé par l'hôte, vous pouvez appuyer sur F4 pendant la partie pour consulter les statistiques du match : le Délai est marqué en bas à droite.

Pendant la partie :

  • si une tortue apparaît en haut à gauche : cela signifie que le jeu attend un paquet venant du PC distant et qu'éventuellement, ce paquet a été perdu
  • si un câble barré d'une croix rouge apparaît : cela signifie qu'un ou plusieurs paquets ont été perdus

Protection du nom :

Les utilisateurs ayant acheté la version complète du jeu ont leur pseudo online protégé. Si quelqu'un d'autre tente de prendre le même nom, le préfixe "[Fake] " sera ajouté devant.

ELO :

Le score ELO donne une approximation de votre niveau de jeu comparé aux autres joueurs. Son but est de vous aider à trouver d'autres joueurs qui ont plus ou moins votre niveau, afin de faire des matchs plus intéressants. Son but n'est pas de déterminer qui est le meilleur joueur de Tennis Elbow, car il est beaucoup trop imprécis. Vous devez jouer autour de 50 à 100 matchs afin d'avoir un score relativement précis.

Pénalité ELO :

Si vous abandonnez le match avant la fin de la partie, vous aurez une pénalité à votre ELO de 3%. Mais si pendant plus d'une minute vous adversaire bascule sur une autre application ou reste dans les menus ou regarde le replay, alors vous pouvez abandonner sans avoir de pénalité, si vous avez passé cette minute dans la partie, sans ouvrir les menus, ni basculer vers une autre application, ni regarder le replay.

Désynchronisation :

Parfois, quand vous jouez en ligne, vous verrez le message "Désynchronisation détectée". Cela veut dire que votre partie et la partie de votre adversaire ont perdu leur synchronisation, et des choses différentes se sont produites sur votre ordinateur et le sien. Dans ce cas, les parties seront resynchronisées, et vous devrez rejouer le point.

Si jamais vous avez beaucoup de désynchronisation (plus d'une tous les 5 points), cela peut vouloir dire que vous et/ou votre adversaire avait installé un Mod qui change la façon du moteur du jeu de fonctionner, ou que votre installation a été corrompue. Dans ce cas, vous et/ou votre adversaire devez enlever tous les Mods installés et redémarrer Tennis Elbow, ou désinstaller puis réinstaller Tennis Elbow, afin de pouvoir jouer à nouveau en ligne normalement.

 

 IV. Jeu 

Coups disponibles :
(b1 = bouton n°1; b2 = bouton n°2)

  • b1 = Coup normal
  • b1 + up = Balle courte
  • b1 + down = Lift
  • b2 = Slice
  • b2 + up = Amortie
  • b2 + down = Lob de défense
  • b1+b2 = Accélération
  • b1+b2 + up = Accélération courte
  • b1+b2 + down = Lob d'attaque
  • pas de bouton = Coup assuré

Pendant le service :

  • b1+b2 = 1er service à vitesse maximale
  • b1 = 1er service assuré, mais plus lent
  • b2 = 2nd service, 100% sûr
  • haut = Ajouter de l'effet slicé
  • bas = Ajouter de l'effet lifté

Le haut et le bas sont inversés lorsque votre joueur se trouve en haut de l'écran.

Mimiques :

A la fin du point, vous pouvez faire une mimique en pressant b1+b2, et haut, bas, gauche ou droit ;

  • si vous avez gagné le point :
    • haut : saut
    • gauche : poing d'encouragement
    • droit : poing rageur
    • bas : excuse de la main
  • si vous avez perdu le point :
    • haut : applaudir votre adversaire
    • gauche : se taper le front avec la raquette
    • droit : fou!
    • bas : jeter votre raquette

Notes :

  • si votre joueur est gaucher, alors la gauche et la droite sont inversées
  • en fonction du n° de coup droit choisi, certaines mimiques peuvent être différentes (mais gardent le même sens)

Comment joue-t-on ?

Déplacement, frappe et visée :

Quand on n'appuie sur aucun bouton, on déplace le joueur dans toutes les directions.
Quand on appuie sur un bouton le joueur se déplace lentement vers la position idéale, plus longtemps on met vers la gauche (ou la droite), plus la balle partira vers la gauche (ou la droite).

==>> Il ne faut pas lâcher le(s) bouton(s) quand on frappe dans la balle !

Quand on ne donne pas de direction, la balle part vers le centre du terrain adverse.

Plus vous êtes près du filet, plus la visée part rapidement vers la gauche ou la droite. Ainsi, il ne faut qu'une fraction de seconde pour envoyer la balle sur la ligne quand on est à la volée, alors qu'il faut 2 ou 3 fois plus de temps quand on est plusieurs mètres après la ligne de fond du court.

A la volée, quand la balle passe loin de votre joueur, appuyez sur le bouton afin de faire un saut vers la balle.

Précision des coups :

Plus vous appuyez sur le bouton à l'avance, plus votre coup sera précis. En effet, si vous n'appuyez qu'au tout dernier moment, votre balle pourra partir dans une large zone, alors que si vous appuyez 1 seconde à l'avance, la zone sera nettement réduite.

Vous pouvez voir la zone de visée et de précision quand votre "Preview" est réglée sur "Tout + Zone de Danger + Visée". Dans ce cas, vous verrez en temps réel l'endroit où vous visez, ainsi que la zone dans laquelle la balle peut partir.

Longueur des coups :

Plus vous préparez tôt, plus :

  • vos coups courts (y compris l'amortie) seront courts
  • vos autres types de coups seront longs

Mais si vous ne centrez pas la balle ou que vous êtes mal placé, vos coups courts seront plus longs, et les autres coups seront plus courts.

Zone de Danger :

Au tennis, quand la balle est montante, juste après le rebond, elle est plus dure à frapper, surtout quand elle est rapide. La zone où il est dangereux de frapper la balle est affichée quand votre "Preview" est réglée sur "Tout + Zone de Danger" ou plus.

Si vous frappez la balle alors que vous vous tenez dans la partie rouge de la "Zone de Danger", vous risquez de perdre le contrôle de la balle et de l'envoyer soit dans le filet, soit trop longue.

Si vous vous tenez dans la partie orange, alors vous risquez de perdre de la précision sur votre coup, et vous verrez la zone de visée s'agrandir (si vous avez réglé "Preview" sur "Tout + Zone de Danger + Visée").

En plus, être placé dans la zone rouge ou orange vous fera perdre de la puissance sur votre coup, tout spécialement lorsque vous essayez de faire des accélérations.

Centrer la balle :

Au tennis, il est très important de bien frapper la balle avec le centre de la raquette, sur le côté de votre corps. Si vous essayez de frapper la balle alors qu'elle vous arrive sur le corps, il est beaucoup plus difficile de faire un bon coup.

Pour cette raison, quand vous souhaitez faire une accélération, une amortie ou une volée, il faut bien faire attention à votre placement, sinon vous pourrez perdre le contrôle de la balle, ou perdre de la précision, selon l'étendue de votre erreur de placement, de la même manière que lorsque vous vous placez dans la partie rouge de la "Zone de Danger".

Mais mal centrer la balle apporte également des malus sur les autres coups : généralement, cela vous fera jouer plus court si vous essayez de faire une balle normale, et plus long si vous essayez de faire une balle courte. Vous pouvez également perdre de la puissance dans votre coup, spécialement lorsque vous faîtes une accélération.

Lorsque vous ratez un coup, en sortant la balle ou en l'envoyant dans le filet, une icône avec un joueur tapant dans la balle apparaît dans un coin de l'écran pour vous montrer si vous bien ou mal centré votre balle :

  • si le fond de l'icône est rouge, cela veut dire que vous avez frappé la balle avec votre corps
  • si le fond est orange, cela veut dire que vous avez moyennement décentré la balle
  • si le fond est vert, cela veut dire que vous avez bien centré la balle

Lob & Amortie :

Le lob et l'amortie s'effectuent comme les autres coups, mais si jamais vous tentez de les effectuer sur une balle qui est trop rapide, alors vous en perdrez totalement le contrôle, et vous ferez un coup beaucoup plus long que prévu.

De plus, lorsque vous effectuez un lob en courant trop vite, vous envoyez la balle dans la même direction que votre course.

Contrôles Arcade, Simulation & Elite :

Par défaut, les contrôles sont en mode Simulation, mais si vous trouvez le jeu trop dur, si vous faîtes trop souvent des fautes directes, alors vous pouvez passer les contrôles en mode Arcade, en cliquant sur l'icône dans la Feuille de Personnage de votre joueur. Dans ce cas, les malus (perte de précision et de puissance) dus aux erreurs de placement dans la "Zone de Danger" et de centrage de balle sont divisées par 2. De plus, la vitesse à laquelle la zone de visée devient précise est multipliée par 2, donc au bout d'une demi-seconde de préparation de votre coup en mode Arcade, vous aurez atteint le même niveau de précision qu'au bout d'une seconde de préparation en mode Simulation. Choisir de jouer avec le mode Arcade n'influence pas les joueurs contrôlés par l'ordinateur.

De manière opposée, si vous trouvez le jeu trop facile, ou si vous voulez avoir plus de fautes pendant la partie, vous pouvez passer en mode Elite. Dans ce mode les effets de la Zone de Danger sont plus importants, et la précision de la zone de visée est abaissée. Quand vous passez en mode Elite pour votre joueur, alors les joueurs ordinateurs passent également dans ce mode, afin d'être à armes égales pendant le match.

Service :

Le service s'effectue de la même manière que les autres coups, à la différence que quelle que soit la combinaison des boutons, une poussée vers le haut provoque un slice et une poussée vers le bas provoque un lift. Vous pouvez déplacer votre joueur sur la gauche ou la droite afin de trouver un angle spécial pour votre service.

Vous pouvez également relâcher le(s) bouton(s) légèrement avant de taper dans la balle sur votre service ; pour le service volée, cette technique vous permet d'atteindre le filet plus facilement ; pour utiliser cette possibilité, relâchez le(s) bouton(s) pressé(s), puis relâcher la direction (si besoin est) puis appuyez sur haut (+ gauche/droit si besoin est) ; cela marche aussi pour reculer, mais dans ce cas, l'efficacité est nettement moindre.

Score :

Le score du joueur 1 est inscrit dans les cases du haut du tableau de score, et celui du joueur 2 en bas. Si la petite balle est dans la case du haut, alors c'est au joueur 1 de servir, sinon c'est au joueur 2.

Trace de balle :

Si dans votre feuille de personnage "Preview" est sur "Service" ou plus, la trace de l'impact de la balle au sol au moment du service est marquée un peu à l'avance afin de vous aider à anticiper la trajectoire.
Si "Preview" est sur "Service + Echanges" ou plus, la trace de l'impact de la balle au sol pendant l'échange est marquée un peu à l'avance afin de vous aider à anticiper la trajectoire.

Accélération à hauteur d'épaule :

Quand vous faites une accélération sur une balle qui se trouve à peu près à hauteur de votre épaule, vous obtenez un bonus sur la vitesse de votre coup. Ce bonus est plus ou moins grand en fonction de :

  • votre style de joueur (reportez-vous à la section "Style du joueur" pour plus de détails)
  • la différence de hauteur depuis votre épaule, à part pour le revers à 2 mains dont le point idéal se trouve un peu plus bas, et le revers à 1 main dont le point idéal se trouve encore un peu plus bas
  • votre temps de préparation : le plus longtemps, le mieux c'est
  • la balle ne doit plus être en phase ascendante, ou alors elle doit avancer très lentement
  • vous frappez à 1 ou 2 mains, ce qui donne du plus rapide au plus lent : coup droit à 1 main, à 2 mains, revers à 2 mains, à 1 main

Fatigue:

Si dans la feuille de personnage la "Fatigue" est désactivée, alors votre joueur se fatiguera uniquement lorsqu'il fait des accélérations.

Vous avez un nombre limité de bonnes accélérations par point, qui dépend de votre pourcentage d'endurance. Une fois que vous les avez toutes utilisées, vous frapperez la balle avec de moins en moins de précision et de puissance à chaque que vous ferez une accélération supplémentaire. Lorsque vous effectuez votre dernière bonne accélération et les suivantes, de la fumée s'échappe de votre raquette pour vous l'indiquer.

Si dans la feuille de personnage la "Fatigue" est activée, alors votre joueur se fatiguera lorsqu'il tape dans la balle et lorsqu'il court.

Pendant le match, vous verrez 2 barres à l'écran pour vous aider à surveiller votre fatigue :

  • Barre du haut = Energie Instantanée
  • Barre du bas = Energie Totale

Les mécanismes de base sont :

  • L'énergie instantanée descend et remonte assez rapidement
  • L'énergie totale descend nettement plus lentement et remonte encore plus lentement
  • L'énergie instantanée ne peut pas remonter plus haut que le niveau actuel d'énergie totale

Vous perdez de l'énergie quand :

  • vous courez ; le plus votre Vitesse est au dessus de votre Endurance, le plus rapidement vous perdez votre énergie ; le plus votre Endurance est au dessus de votre Vitesse, le moins vite vous perdez votre énergie ; ainsi, courir à votre vitesse maximale avec 100% en Endurance & Vitesse vous fatiguera autant que courir à votre vitesse maximale lorsque ces 2 compétences sont à 0%
  • vous frappez des accélérations ; le plus haut votre Endurance est, le moins d'énergie vous perdez quand vous faites des accélérations ; si vous avez un style de jeu qui donne un nombre d'accélérations supplémentaires (par exemple : Attaquant de fond de court), alors vous serez un peu moins fatigué par vos accélérations
  • vous frappez un coup normal ; mais c'est nettement moins fatiguant que de faire une accélération ; mettre beaucoup de lift dans la balle est aussi plus fatiguant

Le plus votre barre d'Energie Instantanée descend en dessous de 20%, le plus :

  • vous perdez de la précision sur vos coups, et plus particulièrement sur vos accélérations
  • vous perdez de la puissance sur vos accélérations, mais pas sur les autres coups
  • votre vitesse maximale ainsi que votre accélération de course diminuent

Le plus votre barre d'Energie Totale descend en dessous de 50%, le plus :

  • vous perdez un peu de précision sur vos coups
  • votre vitesse maximale ainsi que votre accélération de course diminuent

Vous récupérez de l'énergie :

  • un peu après chaque point
  • un peu plus au changement de côté, qui se produit tous les 2 jeux ; il n'y a cependant pas de temps de repos après le 1er jeu de chaque set, donc à ce changement de côté, la récupération est moindre
  • encore un peu plus à la pause en fin de set, qu'il y ait ou non un changement de côté

Quand votre barre d'énergie instantanée est en dessous de 25%, vous en récupérez pendant le point si vous ne bougez pas, mais cela remonte très lentement.

Notes :

  • si vous diminuez le nombre de jeux par set, alors votre Energie Totale diminuera plus rapidement, afin de simuler un match en 6 jeux par set
  • les coups sont un peu plus fatiguant quand la puissance est supérieure à l'endurance, et un peu moins fatiguant quand la puissance est en dessous de l'endurance, spécialement pour les accélérations

Influence des caractéristiques :

Coup Droit & Revers :

Plus vous avez une puissance élevée, plus vous frappez fort dans la balle, spécialement sur le coup normal, le lift, et les accélérations.

Plus vous avez une régularité élevée :

  • plus vous pouvez frapper la balle dans une situation difficile (balle basse, rapide ou montante) sans pénalité de précision
  • plus vos coups normaux seront longs, et plus vos coups courts seront courts

La compétence de précision est relative à celle de puissance. Par exemple avoir 60% en puissance et 70% en précision vous donnera autant de précision dans vos coups que d'avoir 30% en puissance et 40% en précision.

Ainsi plus votre précision est supérieure à votre puissance, moins vous avez besoin de préparer votre coup à l'avance pour qu'il soit précis. Et plus votre précision est inférieure à votre puissance, plus vous avez besoin de préparer votre coup à l'avance pour qu'il soit précis.

Avoir la précision supérieure à la puissance augmente également l'efficacité et la vitesse du coup court et de l'accélération courte.

Coups à 1 main contre coups à 2 mains :

Comparés à leur version à 1 main, le coup droit et le revers à 2 mains sont :

  • un peu moins puissants : vous donnez un peu moins de vitesse à la balle
  • un peu moins amples : vous avez un peu moins d'envergure avec, vous devez donc être plus proche de la balle pour exécuter votre coup
  • un peu plus sûrs : vous avez un malus plus petit quand vous frappez la balle en vous tenant dans les zones rouges et oranges de la Zone de Danger, ainsi vos coups sont souvent un peu plus précis
  • meilleurs pour contrer les coups puissants : cela vous donne un bonus de 15 points à votre compétences en contre

Service :

Comme pour le coup droit et le revers, la précision au service est relative à sa puissance : avoir 80% en puissance et 70% en précision vous donnera autant de précision que d'avoir 50% en puissance et 40% en précision.

Lorsque vous servez en appuyant sur b1+b2, vous tentez de faire votre service le plus rapide possible. Si votre puissance et votre régularité sont égales, alors vous avez 70% de chance de le réussir. Si votre puissance est supérieure à votre régularité, alors vous avez moins de 70% de chance de réussite, et dans le cas inverse, plus de 70% de chance de réussite.

Lorsque vous servez en appuyant sur b2, vous avez 100% de chance de réussite, et la vitesse de votre service dépend de la puissance et du rapport entre la puissance et régularité.

Lorsque vous servez en appuyant sur b1, vous faites un service entre celui de b1+b2 et celui de b2. Si votre puissance et votre régularité sont égales, alors vous avez 85% de chance de le réussir.

Volée & Smash :

En fonction du coup que vous effectuez, la compétence de présence filet est combinée avec la compétence de volée coup droit, volée de revers ou smash, afin de déterminer la précision et la régularité de votre coup. La puissance quant à elle ne dépend pas de la présence filet.

Hormis cela, les volées et les smashs fonctionnent comme le coup droit et le revers du fond du court.

D'autre part, plus votre présence filet est élevée, plus vous pouvez frappez la balle en courant sans avoir de malus.

Lorsque que vous jouez à la volée, vous avez besoin d'avoir des compétences de volée élevées si vous voulez pouvoir faire des volées gagnantes qui débordent votre adversaire. De plus, le coup normal à la volée ne correspond qu'à une poussée de balle, qui souvent sera trop lente pour être hors de portée de votre adversaire, même si elle est bien placée. Si vous voulez faire des volées gagnantes, vous devez utiliser l'accélération, mais uniquement sur les balles faciles, c'est-à-dire pas trop hautes, pas trop basses et pas trop rapides, au risque sinon de faire une faute.

Lob & Amortie :

Les compétences de lob et d'amortie déterminent votre habilité à effectuer respectivement le lob et l'amortie, mais chaque compétence est pondérée avec vos compétences de coup droit ou de revers, en fonction que vous effectuez un coup droit ou un revers.

Plus cette compétence pondérée est élevée, plus vous pouvez garder le contrôle de la balle lorsqu'elle arrive rapidement, et effectuez votre coup normalement, sans que votre coup n'aille trop long.

Contre :

Le contre est la capacité à utiliser la vitesse actuelle de la balle pour la renvoyer plus vite. Le contre vous donne donc un bonus de vitesse sur le coup normal et le lift lorsque la balle arrive vraiment vite, pouvant aller jusqu'à 7,5km/h.

De plus, plus votre compétence en contre est élevée, moins vous êtes gêné par les balles rapides, mais moins les balles lentes sont faciles pour vous également.

Lift :

Le lift est l'effet de bas en haut donnée à la balle par la raquette au moment de la frappe, ce qui la fait tourner rapidement sur elle-même. Les balles liftées ont des trajectoires plus bombées, avec un rebond plus haut et plus rapide qui force vos adversaires à reculer. Mais attention, le lift n'a que très peu d'effet sur surface rapide, tout particulièrement sur herbe.

Plus votre compétence en lift est élevée, plus vous lifterez vos coups, surtout le coup normal, le coup lifté et l'accélération. Une balle liftée donne plus de précision en longueur, mais par contre, elle perd en vitesse pure.

Vitesse :

Plus le pourcentage en "Vitesse" est élevé, plus :

  • le joueur courra vite
  • le joueur démarrera sa course vite
  • le joueur fera de grands bonds à la volée

Taille :

Plus votre taille est petite, plus vous démarrerez votre course rapidement. Mais ce bonus est très petit, juste assez grand pour compenser la différence d'envergure avec les joueurs plus grands, afin de pouvoir jouer avec un joueur plus petit sans être désavantagé.

Style du joueur :

En fonction du style de joueur que vous choisissez, vous aurez certains bonus et malus.

  • Défenseur :
    • nombre de lobs d'attaque illimité
    • 1 accélération supplémentaire
    • bonus de lift sur le coup lifté
    • malus de profondeur et de vitesse plus faibles sur le coup lifté
    • faible bonus de vitesse pour les accélérations au hauteur d'épaule
  • Attaquant de fond de court :
    • 3 accélérations supplémentaires
    • grand bonus de vitesse pour les accélérations au hauteur d'épaule
  • Puncheur :
    • 2 accélérations supplémentaires
    • bonds légèrement plus grands à la volée
    • grand bonus de vitesse pour les accélérations au hauteur d'épaule
  • Polyvalent :
    • 1 accélération supplémentaire
    • bonds plus grands à la volée
    • grand bonus de vitesse pour les accélérations au hauteur d'épaule
  • Volleyeur :
    • bonds nettement plus grands à la volée
    • moyen bonus de vitesse pour les accélérations au hauteur d'épaule
  • Contreur :
    • nombre d'accélération normale illimité, mais puissance nettement diminuée, particulièrement sur les balles lentes ; la compétence de contre est utilisée pour donner de la puissance supplémentaire en fonction de vitesse de la balle avant la frappe
    • certains effets de la compétence Contre augmentés
    • peut prendre la balle plus tôt après le rebond avec moins de pénalités
    • bonds plus grands à la volée
    • faible bonus de vitesse pour les accélérations au hauteur d'épaule

Note: quand la Fatigue est activée, le nombre d'accélérations supplémentaires se traduit par moins d'énergie perdue quand vous faites des accélérations ; de plus, le style Contreur voit ses accélérations être aussi peu fatiguantes que ses coups normaux.

Coup droit & revers à 2 mains :

Comparés à leur version à 1 main, ces coups ont ces bonus & malus :

  • diminue les malus quand le joueur frappe la balle dans la Zone de Danger
  • augmente les malus quand les balles est trop haute ou trop basse
  • donne un bonus de 10 points à la compétence de Contre
  • donne un malus de 10% à la puissance (exemple : 100% de puissance devient 90%, et 50% devient 45%)

Preview :

  • Aucun : ne voit aucun impact de balle à l'avance
  • Service : voit les impacts de balle à l'avance au service
  • Service + Echanges : voit les impacts de balle à l'avance au service et pendant les échanges
  • Tout + Zone de Danger : idem que Service + Echanges, mais affiche en plus la Zone de Danger
  • Tout + Zone de Danger + Visée : idem que précédemment, mais rajoute l'indicateur de visée et de précision

Niveau & Sous-Niveau :

Règlent la force générale de l'ordinateur. Le sous-niveau vous permet de régler finement sa force entre 2 niveaux. Si vous mettez le sous-niveau à 1, alors il sera juste un tout petit peu plus fort qu'en étant au niveau inférieur avec le sous-niveau à 10.

Aux niveaux Débutant, Club & Junior, l'ordinateur a des malus sur ses compétences. Aux niveaux Pro et Master, ses compétences fonctionnent exactement comme les vôtres. Au niveau Incroyable, il a un bonus sur ses compétences.

Le niveau Pro est conçu pour être le niveau de difficulté apportant le plus de réalisme. Mais choisissez le niveau avec lequel vous vous amusez le plus, car c'est ça le plus important..!

Ecran de statistiques :

Cet écran montre la force des coups d'un joueur, ainsi que le nombre de balles qu'il a frappées, le nombre de balles qui ont été dans le court, et le détails des points gagnants; on peut y voir de plus le nombre d'aces effectués, le nombre de doubles fautes, et la distance parcourue.
Le classement du joueur n'apparaît que dans le mode "Compétition".
Si on accède à cet écran pendant un entraînement ou un match, les statistiques affichées concernent uniquement le match; si on y accède par la feuille de personnage, les statistiques concernent la carrière du joueur.

Double :

Au cours d'un double, le joueur n°1 fera équipe avec le joueur n°3, et le joueur n°2 fera équipe avec le joueur n°4.

 

 V. Mode tournoi 

Dans le mode Compétition, les femmes ont leur puissance et leur vitesse diminuée de 25 points comparée aux hommes. Par exemple, si une femme a 75% en puissance en coup droit, alors elle frappera la balle aussi fort qu'un homme ayant 50% de puissance en coup droit.

Sauvegardes automatiques :

Lorsque vous quittez le jeu en cliquant sur le bouton "Quitter", votre partie est sauvegardée automatiquement, et vous reprendrez automatiquement au point où vous en étiez la prochaine fois que vous lancerez le jeu.

De plus, le jeu sauvegarde votre partie automatiquement sur les 3 dernières semaines de jeu, dans 3 emplacements différents ; ainsi, en cas de problème (par exemple, si le jeu plante pour une raison ou une autre), vous aurez toujours la possibilité de reprendre une sauvegarde récente.

/!\ Attention : quand vous jouez en mode fenêtré, et que vous quittez en cliquant sur la petite croix en haut à droite de la fenêtre de jeu, votre partie en cours sera également sauvegardée automatiquement si vous êtes dans un menu disposant du bouton "Quitter" ; par contre, si ce bouton est absent, il se peut que votre partie ne soit pas sauvegardée ; dans ce cas, vous perdrez votre progression depuis la dernière sauvegarde automatique.

Type of carrière :

quand vous démarrez une nouvelle partie, vous avez le choix entre 2 modes :

  • Saisons
  • Carrière

Les 2 modes sont très similaires, mais le mode Saisons contient 3 différences majeures :

  • vous ne pouvez pas créez votre propre joueur, vous devez en choisir un dans la base de joueurs (mais vous pouvez cependant l'éditer)
  • vous pouvez démarrer plus vieux que 17 ans, et à la fin de chaque saison, le jeu vous demandera si vous désirez continuer ou arrêter
  • vos compétences ne diminueront pas et vous ne gagnerez aucun point d'expérience, donc vous n'aurez pas besoin d'entraîner vos compétences ; celles-ci seront changées à chaque début de saison (si vous l'acceptez), afin de refléter la force du joueur que vous avez choisi pendant cette nouvelle année à venir

Classement pour les hommes :

  • Le nombre de points pour le classement technique est la somme des 18 meilleurs tournois joués, sur les 52 semaines précédentes, comprenant obligatoirement les 4 Grand Chelems plus les 8 meilleurs des 9 Master Series auxquels le joueur se devait de participer si son classement lui permettait d'entrer dans le tableau final ; de plus, les joueurs du Top30 peuvent compter leurs 4 meilleurs TET500 et leurs 2 meilleurs TET250 ; les joueurs inférieurs peuvent compter jusqu'à 4 TET250
  • Le nombre de points pour la Race est la somme des 18 meilleurs tournois joués, depuis le début de l'année, avec les mêmes règles pour les tournois requis que le classement technique

Classement pour les femmes :

  • Le nombre de points pour le classement technique est la somme des 16 meilleurs tournois joués en Simple, et les 11 meilleurs en Double, sur les 52 semaines précédentes, comprenant obligatoirement tous les tournois majeurs auxquels la joueuse se devait de participer si son classement lui permettait d'entrer dans le tableau final
  • Le nombre de points pour la Race est la somme des 16 meilleurs tournois joués en Simple, et les 11 meilleurs en Double, depuis le début de l'année, comprenant obligatoirement tous les tournois majeurs auxquels la joueuse se devait de participer si son classement lui permettait d'entrer dans le tableau final
  • Les tournois majeurs sont les 4 tournois du Grand Chelem, plus les tournoi PremiumM et les 2 meilleurs résultats en Premium5

Entrainement :

En mode carrière, avant chaque tournoi, apparaît l'écran d'entraînement; cet écran vous permet de distribuer les points d'expérience que vous avez gagnés lors de vos précédents matchs.

Pour chaque jour disponible entre 2 tournois, vous avez 6 heures d'entraînement. Selon le mode de difficulté, les heures d'entraînements sont plus ou moins efficaces. De plus, il faut 2 fois plus de temps et d'expérience pour entraîner la plupart de vos compétences entre 60% et 89%, et 3 fois plus de temps et d'expérience quand elles sont entre 90% et 99%.

Dans les tableaux finaux, pour chaque match remporté, vous gagnez environ 10 à 20 points d'entraînement, pour chaque match perdu, vous gagnez 25 points d'entraînement, car on apprend plus de ses défaites que de ses victoires..! Ce nombre de points est divisé par 2 dans les qualifications. De plus, dans les niveaux les plus faciles (Débutant & Club) vous gagnerez plus de points, et dans les niveaux les plus difficiles (Maître et Incroyable), vous gagnerez moins de points.

Mais attention, toutes les semaines, vous perdez quelques points dans vos compétences : si vous ne vous entraînez pas, vous régresserez inexorablement. Le plus haut est une compétence, le plus vite elle baissera.

Le lift est une compétence spéciale, car c'est la seule que vous pouvez choisir de faire baisser. En effet, sur surface rapide, le lift n'est pas très utile et donc il peut être intéressant d'avoir un joueur de type volleyeur avec un lift très bas.

Bonus de fin d'année : si vous ne jouez pas durant la dernière semaine de Décembre (attention: cela exclut les qualifications des tournois se déroulant la 1ère semaine de l'année suivante !), vous obtiendrez un petit bonus d'expérience, dépendant du temps pendant lequel vous n'avez pas joué, limité à 50 jours maximum ( = depuis la mi-Novembre), pour un total au maximum de ~35-45 points d'expérience, en fonction du niveau de difficulté.

Note : si vous gagnez un tournoi, c'est toujours un dimanche. Et si vous commencez un nouveau tournoi la semaine suivante, souvent, ce sera le lundi : vous n'aurez donc aucun jour libre pour l'entraînement, c'est pourquoi vous verrez 0 jour & heure d'entraînement disponible dans l'écran d'entraînement.

Les Masters :

Vous participerez automatiquement aux Masters si vous êtes dans les 8 premiers joueurs mondiaux au moment où ce tournoi a lieu, ou si vous avez gagné un tournoi du Grand Chelem. Si ce n'est pas le cas, vous n'y jouerez pas.
Son déroulement est un peu spécial. Il y a 2 poules de 4 joueurs, et chaque joueur fait 3 matchs. Les 2 meilleurs joueurs de chaque poule vont alors en demi-finales, puis les 2 vainqueurs des demi-finales se rencontrent en finale.

Changer les joueurs du mode tournoi :

 Eh ! Les Cheat Codes ??? 

Ok, mais utilisez-les avec sagesse !

  • mtkil : gagner le match
  • mtwin : gagner le set actuel
  • mtbal : grossir la balle
  • mtrun : courir plus vite
  • mtbozo : Bozo le clown ! :-p
  • '²' + Maj de droite + '-' du pavé numérique, en même temps : gagner tous les points
  • Ctrl de gauche + Alt de gauche + '0' du pavé numérique : arrêter de gagner tous les points
    (touches sur un clavier AZERTY)
  • Coups secrets :
    • pour effectuer un coup lifté avec effet sur le côté : appuyez sur Coup Normal, puis Coup Court, puis Coup Lifté (par défaut => B1, puis B1+haut, puis B1+bas)
    • pour effectuer un slice court : appuyez sur Slice, Amortie, Slice ( => B2, puis B2+haut, puis sur B2)
    • pour effectuer un slice de défense calamiteux : appuyez sur Slice, Lob de Défense, Slice ( => B2+bas, puis sur B2)
